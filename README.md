# Интерполяция функций полиномами

В корневой директории команда `make` создаст исполнимые файлы: `polyinterp` и `tests`.

## Основная программа `polyinterp`

Использование:
````
```bash
./polyinterp <chebyshev|uniform> [q]
```
Первый аргумент должен принимать одно из двух значений: `chebyshev` и `uniform`
для запуска интерполяции на Чебышевской и равномерной сетках соответственно.
Второй необязательный аргумент `q` -- целое число, задающее значение `q`. (По умолчанию `q = 100`)
Команды `make chebyshev` и `make uniform` запустят программу для соответствующих сеток при `q = 100`,
и построят график интерполирующего полинома в файле `plot_result.pdf`.

## Программа `tests`

Выполняет интерполяцию заданных в программе функций,
затем записывает результаты в файлы вида `res_[название функции]_[тип сетки].dat`. 
После получения всех файлов запускает Gnuplot-скрипт, который создает файл `test.pdf`.
В нем сравниваются функции, вычисленные через Gnuplot,
и полученные интерполяцией на разных сеткaх.
